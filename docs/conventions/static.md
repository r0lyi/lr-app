# Static

- `static/` global: foundation visual, componentes reutilizables, layouts y branding.
- `apps/<app>/static/<app>/`: CSS/JS específicos de páginas o interacciones de la app.
- Naming recomendado:
  - `css/pages/<screen>.css`
  - `js/pages/<screen>.js`
- Si un asset se usa en más de una app, debe promocionarse a `static/`.
